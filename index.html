💰 總預算估計

  • 交通+住宿： 約20,000-30,000元
  • 用餐+景點： 約15,000-25,000元
  • 購物+雜費： 建議預留10,000-20,000元
  • 總計： 45,000-75,000元（不含機票）

✈️ 航班資訊

🛫 去程

航班： 中華航空 CI220

時間： 09:00 → 13:10

機場： 台北松山 (TSA) → 東京羽田 (HND)

🛬 回程

航班： 中華航空 CI221

時間： 14:30 → 16:55

機場： 東京羽田 (HND) → 台北松山 (TSA)

上機注意

  1. 托運行李內請勿置放貴重物品及鋰電池。
  2. 指甲刀、剪刀等刀具類請置放於托運行李。
  3. 乳液狀物品隨身攜帶，每瓶限100CC，並置於透明夾鍊袋。

🏩 住宿資訊

品川王子大飯店 (Shinagawa Prince Hotel) Main Tower - 位置便利，近品川車站。

💳 旅行支付方式建議

  1. 行動支付 (玉山Wallet+PayPay):
    直接扣日圓帳戶，享回饋，適用範圍廣。
  2. 交通卡 (Suica/ICOCA):
    方便搭乘交通及小額支付。
  3. 信用卡 (玉山熊本熊卡):
    適合指定通路高單價購物。
  4. 信用卡 (聯邦吉鶴卡):
    適用於一般消費。
  5. 現鈔 (日圓):
    備用，適用於小店鋪。

其他資訊

👨‍👩‍👧‍👧 額外建議

  • **家庭成員考量：** 兩個女兒（10歲/4歲）的年齡差異較大，行程安排上需兼顧兩者的興趣。
  • **彈性行程：** 若孩子疲憊或對某景點特別感興趣，可適時調整行程。
  • **緊急情況：** 隨身攜帶護照影本、旅遊保險資訊，並記下台灣駐日代表處的電話。

🚨 應急與替代方案

  • **天氣不佳或預定失敗：** 可將戶外行程調整為室內購物中心或景點（如東京鐵塔、晴空塔）。
  • **交通延誤：** 東京的鐵路系統非常發達，若JR線延誤，可快速查詢Google Maps尋找替代的地鐵路線。

🏥 健康與安全細節

  • **東京當地醫療資源：** 品川車站附近有NTT東日本關東醫院或品川綜合醫院。
  • **最新入境要求：** 出發前務必查詢日本台灣交流協會或日本觀光局的官方網站。
  • **個人衛生：** 隨身攜帶酒精濕紙巾或乾洗手液。
